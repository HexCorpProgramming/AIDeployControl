About this page

This interface is intended to give the QA team control over the QA instance of the HexCorp Mxtress AI. Please always be aware, that there might be others currently using the AI and coordinate accordingly.

Launch the AI

Clicking this button only works, if the AI is not already running.
When you click it, the newest develop version of the AI will be loaded and launched as the QA instance.

Stop the AI

Clicking this button only works, if the AI is currently running.
When you click this button the currently launched QA instance of the AI will be stopped. It may appear to still be logged on to discord, but will not react to commands.

Reset the database

Clicking this button only works, if the AI is not currently running and the database was successfully initialized before.
When you click this button the current database will be removed and it will be reinitialized again when the AI is launched again.
This will remove all data and will for example lead to stored drones being stuck in storage, because there is no more timer to release them.
